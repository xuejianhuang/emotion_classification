# emotion_classification
Emotion classification of online comments based on RNN, the accuracy of the model in the test set reaches 99%
  * **data:** Large Movie Review Dataset.This dataset contains movie reviews along with their associated binary sentiment polarity labels.The dataset contains 50,000 reviews split evenly into 25k train and 25k test sets. The overall distribution of labels is balanced (25k pos and 25k neg). We also include an additional 50,000 unlabeled documents for unsupervised learning.
